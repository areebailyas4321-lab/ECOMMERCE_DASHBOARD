# Ecommerce Dashboard

## Overview
This repository contains a Power BI project designed to analyze Ecommerce sales data. The dashboard provides insights into order trends, sales performance, and other key metrics.

## Files
- **ECOMMERCE.pbit**: The Power BI Template file containing the dashboard layout and data model.
- **Details.csv**: Dataset containing detailed information about individual transactions.
- **Orders.csv**: Dataset containing high-level order information.

## Usage
1. Open `ECOMMERCE.pbit` in Microsoft Power BI Desktop.
2. If prompted, load the data from the provided CSV files (`Details.csv` and `Orders.csv`).
3. Explore the visualizations to gain insights into the ecommerce data.
